# xx2d

this is a simple 2d game engine based by opengl es 3.0 & c++20

performance ~= raylib, much more faster than cocos2dx 3.x

# feature

search path

texture format: pkm

texture cache

texture packer frame cache

quad with auto batch

sprite based of quad

bmfont

label

keyboard, mouse
